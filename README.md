# vim-config
My vim configuration with settings and plugins leveraging [Vundle](https://github.com/VundleVim/Vundle.vim) as git submodule.

## Install Vundle
```zsh
$ cd ~
$ git clone git@github.com:ifthenelse/vim-config.git .vim
$ ln -s .vim/.vimrc
```

## Install plugins
```zsh
$ vim +PluginInstall +qall
```
